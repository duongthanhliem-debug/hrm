⚙️ Cài đặt hệ thống

Cấu hình kết nối Google Sheets, Apps Script và OpenAI

🔌
Kết nối Apps Script
✨
Anthropic API (Claude)
⚠️ Key được lưu trong localStorage · Lấy tại console.anthropic.com
📋
Hướng dẫn cài đặt nhanh
  1. Mở Google Sheet: Click vào đây
  2. Vào Extensions → Apps Script → Dán nội dung file Code.gs
  3. Deploy → New deployment → Web App → Anyone → Copy URL
  4. Dán URL vào ô "Apps Script Web App URL" phía trên
  5. Nhập Anthropic API Key để dùng AI — lấy miễn phí tại console.anthropic.com
  6. Bấm Lưu cài đặt và Kiểm tra kết nối
